# Tomato Taste Test

A triple-elimination tournament app for ranking pizza sauce tomatoes.

## Features

- 6 comparisons to find your top 3 tomatoes
- Auto-saves progress (survives page refresh)
- Mobile-friendly
- CSV export for easy analysis

## How It Works

**Round 1:** Compare 3 groups of 4 cans, pick best and worst from each
**Round 2:** Compare 2 groups of 3 from the survivors, pick best from each
**Round 3:** Rank the final 3 (1st, 2nd, 3rd place)

Total: 21 sauce tastings, 6 comparisons per person

## Usage

1. Enter your name
2. Make comparisons as prompted
3. Copy CSV results and share

Built with vanilla HTML/CSS/JS - no dependencies!
